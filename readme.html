Introduction

How is all life connected? I started thinking about his during my years at University and subsequently applied some of those ideas to startups and those experiences started to formulate the idea of lifestylelinking. In summary, Einstein gave the law of physics, E=mc2, relativity and I was inspired to think that all life is connected and in a relative manner. Gaia theory, James Lovelock also showed how the normalization of different life ecosystems allowed them to be connected and finally 'wisdom of crowds', from the earliest amateur mathematician calculations to the PageRank concept deployed by the Google founders harness the collective intelligence. The lifestylelinking platform pulls together these ideas into one, built on the principals that the individual owns their own data and they have the right to have it work for them, wherever. With the open sourcing of the code, I looking forward to contributing to the wider open source community and to sharing in the discoveries we may find along the way. James

Installation and Use

These instruction focus on the core lifestylelinking processes. The platform uses numerous open source API libraries and installation and user guides for those should be consulted.

  1. Download or unzip files
  2. Create a new mysql database granting appropriate permissions
  3. Edit database details in the following files lifestylelinking/mysqlstart2.php and lifestylelinking/feedreader/config.php
  4. NB. To include videos in the site requires the Zend Gdata Library. To install without video the include file include_once( ABSPATH . 'display/context/youtubedaily.php' ); need deleted from /lifestylelinking/addlogic.php file, note other display html may need edited to remove video display code.
  5. Upload the files to a server.
  6. Navigate to the lifestylelinking/install.php
  7. Next login to the Contropanel at /lifestylelinking/controlpanel/login/login.php
  8. If the installation fails, please delete the all database tables and re-loading install.php
First time use of lifestylelinking
  1. Signin to the Control Panel
  2. First navigate to Definitions Control page and setup a Magazine section(s) and then definition(s), these are created from wikipedia urls ONLY.
  3. Next navigate to Add new data page, add blog urls or use a blog search engine to identify content sources.
  4. Now navigate to the Daily Update page: Set the end date you want for the first results (a day will be automatically added every 24hrs from then on) and then press the buttons in this order: 1. Update qualifying urls, 2. Generate meStats, 3. Generate meLife data and finally 4. Start the processing of daily results.
  5. To add photo go to Daily Flickr photo page and click Update photo (Edit dates if needed)
  6. View results on the site 'relativepath' to /lifestylelinking/index.php
Daily operational routine
  1. Login to the control panel
  2. Navigate to Dailyupdate page.
  3. Update feeds button
  4. Update stats (daily or weekly or as appropriate)
  5. Update melife (daily or weekly or as appropriate)
  6. Create daily results
